当然我不会重造轮子。没那功夫，也没有别人造的好。好了，现在把别人轮子推过来：开源的 highlight.js

URL:https://highlightjs.org

使用方法：

1. 在 html 的 head 里三句话：分别是高光的风格 css ，高光 js 和脚本初始化。

<link rel="stylesheet"
      href="http://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/styles/default.min.css">
<script src="http://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/highlight.min.js"></script>
<script >hljs.initHighlightingOnLoad();</script>

2. 在 body 的 代码块加上 <pre><code> ... </code> </pre>. 脚本自动识别 code 并将其高光。语言类型可以通过 class 显式指定（使用lang-前缀）。

<pre>
<code lang-cpp>
#include < iostream>

int main(int argc, char *argv[]) {

    /* An annoying "Hello World" example */
    for (auto i = 0; i < 0xFFFF; i++)
        cout << "Hello, World!" << endl;

    char c = '\n';
    unordered_map <string, vector<string> > m;
    m["key"] = "\\\\"; // this is an error

    return -2e3 + 12l;
}
</code>
</pre>

其中步骤 1 使用的cdn，也可以自己下载集成。高光风格有很多种，戳这里，下载到指定目录，link进来即可替换。这里有官方的风格示例。
