# xujinguang.github.io
#### 金光博客
## 博客编辑器
	./notepad/notepad.html
## 博客后台服务
	./server/server.py
## 博客生成器
	./server/blog.py
## 安装说明
1. 安装sqlite3
2. 安装python
3. 创建*.db
4. 使用sqlite.sql创建表
5. 填写blog.conf配置
6. 本地终端首先运行服务
```
	$ python server.py
```
7. 打开notepad.html编辑文章
8. 填写必要内容
9. Submit提交，成功则标示本地生成博客成功
10. 博客推送到github
